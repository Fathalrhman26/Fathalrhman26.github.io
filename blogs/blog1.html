My Journey with the GitHub All In Africa Program: Reflections, Ambitions, and the Essence of Open Source

July 29, 2024

Introduction:

My advocacy for social good has always driven me. Being in tech, open source has been my way to contribute while advancing my career and skills. I've followed open source trends for years, looking for ways to contribute. When I found the GitHub All In Africa program, I knew it was the perfect opportunity.

Experience in the Program:

One of the key highlights was contributing to my first open source project it was a major milestone for me. The sessions were incredibly engaging and informative, filled with the enthusiasm of learning and the spirit of collaboration.
Despite my country's current war situation, learning Git and GitHub allowed me and my partner to work on our graduation project remotely, a true lifesaver.

Learning and Development:

I completed all GitHub skills courses, from creating an account to merging pull requests. Learning Git and GitHub has been a game changer, enabling me to overcome remote work challenges and leverage my skills for the greater good.

Personal Growth and Community:

Before joining the All In Africa program, I had a limited understanding of open source. Now, I see it as a collaborative, community-driven movement with immense social impact. The program introduced me to a supportive community and an inspiring regional manager who motivated me to progress. This experience has equipped me with the skills and confidence to pursue a software engineering career.

Open Source and Sustainable Development Goals:

Open source software can be central to achieving Sustainable Development Goals. My passion for open source led me to train as a GitHub DPG open source community manager, where I'll work with AI-based initiatives. I aim to build an open source community in my country, engage more youth in tech, and contribute to rebuilding after the war.

Future Plans and Ambitions:

Post-graduation, I plan to apply my skills in the industry. I'm currently working on a startup focused on AI-driven health and nutrition solutions, which I hope to open source. Additionally, I'll be enhancing my community management skills to better lead open source initiatives addressing SDGs. I aspire to be part of the expert community dedicated to this noble cause, particularly in reinforcement learning and robotics.

Conclusion:

Reflecting on my journey with All In Africa, I'm filled with gratitude for this incredible opportunity. If you're considering exploring open source or joining a program like this, I say—go for it! The experience is life-changing, and the opportunities are endless.

In my next post, I am willing to share abot my journey in entrepreneurship and innovation and how open source promotes creativity and innovation.
